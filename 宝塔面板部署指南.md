# 宝塔面板一键部署指南

## 解决宝塔面板访问问题

### 方法1：使用服务器IP访问
1. 在浏览器中输入：`http://116.62.11.210:8888`
2. 如果显示登录页面，说明可以访问
3. 如果显示访问权限错误，继续方法2

### 方法2：修改宝塔面板配置
1. 使用PuTTY登录服务器
2. 运行命令：`bt 14` （修改面板配置）
3. 选择 "1. 取消域名绑定"
4. 输入 "y" 确认
5. 重启宝塔面板：`service bt restart`
6. 再次尝试访问：`http://116.62.11.210:8888`

### 方法3：检查防火墙
1. 登录服务器
2. 运行命令：`iptables -A INPUT -p tcp --dport 8888 -j ACCEPT`
3. 保存规则：`service iptables save`
4. 重启防火墙：`service iptables restart`

## 宝塔面板部署步骤

### 步骤1：登录宝塔面板
1. 在浏览器中输入：`http://116.62.11.210:8888`
2. 输入用户名和密码登录

### 步骤2：安装必要软件
1. 点击左侧"软件商店"
2. 搜索并安装：
   - Nginx
   - Node.js 18.x
   - PM2管理器

### 步骤3：创建网站
1. 点击左侧"网站"
2. 点击"添加站点"
3. 域名：`wu追情红梦.online`
4. 根目录：选择或创建一个目录（如 `/www/wwwroot/smart-logistics`）
5. 点击"提交"

### 步骤4：上传项目文件
1. 点击刚创建的网站后面的"文件"
2. 点击"上传"，选择本地的项目压缩包
3. 上传完成后，点击"解压"

### 步骤5：安装依赖并构建项目
1. 点击左侧"终端"
2. 进入项目目录：
   ```bash
   cd /www/wwwroot/smart-logistics
   ```
3. 安装依赖：
   ```bash
   npm install
   ```
4. 构建项目：
   ```bash
   npm run build
   ```

### 步骤6：配置PM2
1. 点击左侧"PM2管理器"
2. 点击"添加项目"
3. 项目名称：`smart-logistics`
4. 启动文件：`node_modules/.bin/next`
5. 工作目录：`/www/wwwroot/smart-logistics`
6. 启动参数：`start -p 3000`
7. 点击"提交"

### 步骤7：配置Nginx反向代理
1. 点击左侧"网站"
2. 找到刚创建的网站，点击"设置"
3. 点击"反向代理"
4. 点击"添加反向代理"
5. 代理名称：`nextjs`
6. 目标URL：`http://127.0.0.1:3000`
7. 发送域名：`$host`
8. 启用WebSocket
9. 点击"提交"

### 步骤8：配置域名解析
1. 登录阿里云控制台
2. 进入"域名管理"
3. 找到域名"wu追情红梦.online"
4. 点击"解析"
5. 添加A记录，将域名指向服务器IP：`116.62.11.210`
6. 等待解析生效（约10-30分钟）

### 步骤9：测试访问
1. 在浏览器中输入：`http://wu追情红梦.online`
2. 如果能看到网站页面，说明部署成功

## 常见问题排查
- **宝塔面板无法访问**：按照前面的方法解决
- **网站无法访问**：检查PM2状态和Nginx配置
- **域名解析问题**：检查域名解析记录是否正确
- **应用启动失败**：查看PM2日志

## 查看应用状态
1. 点击左侧"PM2管理器"
2. 查看应用状态
3. 点击"日志"查看运行日志
