## 配置步骤

### 1. Google Analytics 配置

#### 步骤 1: 创建 GA 账号
1. 访问 https://analytics.google.com
2. 登录 Google 账号
3. 点击"开始使用"
4. 创建新账号

#### 步骤 2: 获取跟踪 ID
1. 在 GA 后台，点击左下角 ⚙️ 设置
2. 选择"数据流"
3. 点击"添加数据流" → "网站"
4. 输入网站 URL: `https://smart-logistics-xxx.vercel.app`
5. 复制跟踪 ID (格式: `G-XXXXXXXXXX`)

#### 步骤 3: 在 Vercel 配置
1. 进入 [Vercel Dashboard](https://vercel.com/dashboard)
2. 选择项目 → Settings → Environment Variables
3. 添加变量:
   - **Name**: `NEXT_PUBLIC_GA_ID`
   - **Value**: `G-XXXXXXXXXX` (你的跟踪 ID)
4. 点击 "Save"
5. 重新部署项目

### 2. 百度统计配置

#### 步骤 1: 创建百度统计账号
1. 访问 https://tongji.baidu.com
2. 登录百度账号
3. 点击"添加网站"

#### 步骤 2: 获取跟踪 ID
1. 输入网站信息:
   - 网站名称: 智运物流
   - 网站域名: `smart-logistics-xxx.vercel.app`
   - 网站类型: 物流/运输
2. 提交后获取跟踪代码
3. 找到 `hm.js?` 后面的字符串 (格式: `xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`)

#### 步骤 3: 在 Vercel 配置
1. 进入 [Vercel Dashboard](https://vercel.com/dashboard)
2. 选择项目 → Settings → Environment Variables
3. 添加变量:
   - **Name**: `NEXT_PUBLIC_BAIDU_ID`
   - **Value**: `xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx` (你的跟踪 ID)
4. 点击 "Save"
5. 重新部署项目

## 配置自定义域名（可选）

如果你想使用自己的域名：

### 步骤 1: 购买域名
推荐平台:
- 阿里云: https://wanwang.aliyun.com
- 腾讯云: https://dnspod.cloud.tencent.com
- GoDaddy: https://www.godaddy.com

### 步骤 2: 在 Vercel 添加域名
1. 进入项目 → Settings → Domains
2. 输入你的域名（如 `smartlogistics.com`）
3. 按照 Vercel 提供的 DNS 记录配置域名

### 步骤 3: 配置 DNS
在域名注册商处添加 DNS 记录：

**方式 A: CNAME 记录（推荐）**
```
类型: CNAME
主机: www
值: cname.vercel-dns.com
```

**方式 B: A 记录**
```
类型: A
主机: @
值: 76.76.21.21
```

### 步骤 4: 等待生效
- DNS 生效通常需要 5-30 分钟
- 最长可能需要 24-48 小时
