# 智能物流下单官网一键部署指南

## 准备工作
- 阿里云服务器（IP：116.62.11.210）
- 域名：wu追情红梦.online
- 本地电脑安装：
  - PuTTY（SSH工具）
  - FileZilla（FTP工具）

## 步骤1：登录服务器
1. 打开PuTTY
2. 输入服务器IP：116.62.11.210
3. 端口：22
4. 点击"Open"
5. 输入用户名：root
6. 输入服务器密码（输入时不会显示，直接输入后按回车）

## 步骤2：上传项目文件
1. 打开FileZilla
2. 主机：116.62.11.210
3. 用户名：root
4. 密码：服务器密码
5. 端口：22
6. 点击"快速连接"
7. 在本地站点中找到项目文件夹
8. 将项目文件夹拖拽到远程站点的根目录

## 步骤3：运行部署脚本
1. 在PuTTY中，进入项目目录：
   ```bash
   cd /path/to/smart-logistics
   ```
2. 赋予脚本执行权限：
   ```bash
   chmod +x deploy.sh
   ```
3. 运行部署脚本：
   ```bash
   ./deploy.sh
   ```
4. 等待脚本执行完成（约5-10分钟）

## 步骤4：配置域名解析
1. 登录阿里云控制台
2. 进入"域名管理"
3. 找到域名"wu追情红梦.online"
4. 点击"解析"
5. 点击"添加记录"
6. 记录类型：A
7. 主机记录：@
8. 记录值：116.62.11.210
9. 点击"确定"
10. 等待解析生效（约10-30分钟）

## 步骤5：测试访问
1. 在浏览器中输入：`http://wu追情红梦.online`
2. 或直接访问：`http://116.62.11.210:3000`
3. 如果能看到网站页面，说明部署成功

## 常见问题排查
- **无法连接服务器**：检查服务器IP和密码是否正确
- **脚本执行失败**：检查服务器网络连接
- **域名无法访问**：检查域名解析是否正确配置
- **应用未启动**：运行 `pm2 status` 检查应用状态

## 查看应用状态
```bash
# 查看应用状态
pm2 status

# 查看应用日志
pm2 logs smart-logistics

# 重启应用
pm2 restart smart-logistics
```

## 重启Nginx
```bash
sudo systemctl restart nginx
```

部署完成后，您的智能物流下单官网就可以正常访问了！
